# Common Threads Website

## What This Is
Static HTML/CSS site for Common Threads Consultancy (digital accessibility consulting). No build tools/frameworks — keep it that way unless I say otherwise.

## Conventions
Colors/fonts: Reference the values, don't hardcode new ones unless asked. Use index.html for color value reference.

## Accessibility (non-negotiable)
- WCAG 2.1 AA minimum on everything – this is a consultancy for accessibility, so the site itself must walk the walk
- Every image needs meaningful alt text
- Color contrast must pass AA at minimum
- Keyboard navigation must work everywhere

## Voice/Tone
- Voice should be approachable, warm, direct, knowledgeable, clear, and michigan-rooted.

## What not to do 
- Don't introduce a JS framework or build step
- Don't change the color palette without asking
- Don't restructure existing pages, only fix consistency issues unless asked
- Don't change text wording unless asked.
